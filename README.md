## 📊 Canada Per Capita Income Prediction

This project uses **Machine Learning (Linear Regression)** to analyze and predict Canada’s per capita income based on historical data. The model learns the relationship between year and income, and is used to forecast future income trends.


## 🚀 Features

* Data preprocessing using Pandas
* Linear Regression model using Scikit-learn
* Data visualization of actual vs predicted values
* Future income prediction
* Export of results in submission-ready format
* Plot saving for reporting and analysis


## 🧠 Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn


## 📈 Output

* Model trained on historical income data
* Regression line visualization
* Predicted per capita income for future years
* CSV submission file generated for Kaggle


## 📁 Project Structure

* Data loading & cleaning
* Model training
* Prediction
* Visualization
* Submission file generation


## 🎯 Goal

To understand and apply **simple linear regression** for real-world economic trend prediction.
